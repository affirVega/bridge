# Bridger - мост между Discord, Telegram, VK

Чисто локальный проект по синхронизации чатов друг с другом.

Текущий функционал проекта (после переписывания xD)

|                      | discord | vk  | telegram |
| -------------------- | ------- | --- | -------- |
| Прием событий        |         |     |          |
| - новое сообщение    | ✅       |     |          |
| - редактирование     |         |     |          |
| - удаление           |         |     |          |
| Отправка в свои чаты |         |     |          |
| - новое сообщение    | ✅       |     |          |
| - ответ на сообщение | ✅       |     |          |
| - редактирование     |         |     |          |
| - удаление           |         |     |          |
| Отправка вложений    |         |     |          |
| - Фотография         |         |     |          |
| - Документ           |         |     |          |
| - Стикер             |         |     |          |
| Конфигурация         |         |     |          |
